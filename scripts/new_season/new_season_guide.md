# Guide to Setting Up New XRL Season

## Steps

1. Update CURRENT_YEAR variables
2. Archive user and player stats to previous season and create new rounds (setup_new_season.py)
3. Import new player list (import_player_list.py)
4. Reset stats for players and teams (reset_season_stats.py)
5. Insert new draw (insert_draw.py)
6. Update lambda function code
7. Turn back on scheduled functions in AWS Eventbridge
